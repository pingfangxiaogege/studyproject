https://angular.cn/guide/quickstart
  • 在开始之前，请确保你的开发环境已经包含了 Node.js® 和 npm 包管理器，Angular 需要 Node.js 的 8.x 或 10.x 版本。
  • 要想检查你的版本，请在终端/控制台窗口中运行 node -v 命令。
  • 要想安装 Node.js，请访问 nodejs.org。
  • 安装完node之后，检查npm包管理器是否安装，使用npm -v 命令检查。
  • 全局安装脚手架cli， npm install -g @angular/cli
  • 使用 Angular CLI 来创建项目、创建应用和库代码，并执行多种开发任务，比如测试、打包和发布。
  • 运行 CLI 命令 ng new，并提供一个名字 my-app： ng new my-app
  • ng new 会提示你要把哪些特性包含在初始的应用项目中。请按回车键接受默认值。
    如果需要引入less样式， 需要加上--style less, ng new my-app --style less
  • 现在项目已经创建，进入创建的项目目录cd my-app
  • 进行依赖安装npm i (也可以用yarn， cnpm)
  • 使用ng serve -o 启动项目
  • 到这里项目就创建完毕了
